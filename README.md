# Smart Coffee Roaster Web App
## Features
- Dashboard monitoring for Roasting Session
- Device API :
    - Communication with Smart Coffee Device
        - Start/Stop Roast
        - Get Sensor Data
    - Roasting session :
        - Create/Finish
        - View
- Graph Monitoring :
    - Graph sensor data
    - Line marker for roast status (preheat, charge, ...)
